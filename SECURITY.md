# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |

## Reporting a vulnerability

Please report security issues privately by opening a GitHub Security Advisory on the repository, or by emailing the maintainer directly. Do not open public issues for security vulnerabilities.

We aim to acknowledge reports within 3 business days and provide a fix or mitigation plan within 30 days for confirmed issues.
